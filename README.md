# 🚀 Future Impact Simulator

A minimal productivity simulator that demonstrates how consistency compounds over time.

## 🌐 Live Demo
👉 https://anooshazeeshan.github.io/future-me-simulator/

---

## 💡 About the Project

The Future Impact Simulator helps users understand how daily effort in the form of hours per day and number of days impacts overall progress.

Instead of only showing numbers it provides:
- Growth stage
- Personal insight
- Simple actionable advice

The goal of this project is to make productivity and consistency easier to visualize.

---

## ⚙️ Features

- Takes user input for goal hours per day and number of days
- Calculates total effort
- Shows progress level:
  - Beginner Phase
  - Growth Phase
  - High Momentum Phase
- Gives personalized insight and advice
- Clean and responsive UI
- Reset functionality
